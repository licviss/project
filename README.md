# Telegram Bot for Task Management

This is a Telegram bot for task management using Google Sheets. The bot helps employees track their tasks and materials from a Google Sheets document.

## Requirements

- Python 3.x
- Libraries:
  - `telebot`
  - `gspread`
  - `oauth2client`
  - `gspread-formatting`
